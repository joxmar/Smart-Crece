# Smart Crece 📈

**Empower Your Future: Financial Skills for Every Generation**

Welcome to the **Smart Crece** project! This repository hosts the website and resources for a financial literacy platform designed to help people of all ages—from high school students to retirees—take control of their financial future.

## 🚀 About the Project

Smart Crece is built on the belief that financial freedom begins with understanding. We aim to close the knowledge gap by teaching personal finance, investing, real estate, and entrepreneurship.

**Our Mission:** To empower teenagers, families, professionals, and retirees with the knowledge and confidence to make smarter financial decisions.

### 🎯 Key Audiences & Topics

- **High School & College Students:** Budgeting basics, credit scores, and starting smart.
- **Young Adults:** Managing student loans, saving for big goals (car, home), and investing basics.
- **Families:** Family budgeting, education savings (529 plans), insurance, and estate planning.

## 🛠️ Tech Stack

This project is a static website utilizing modern CSS tooling.

- **HTML5**
- **Tailwind CSS** (v4.0+)
- **NPM** (for package management and scripts)

## 💻 Getting Started

Follow these instructions to set up the project locally.

### Prerequisites

- Node.js and npm installed.

### Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    ```
2.  Navigate to the project directory:
    ```bash
    cd "Smart Crece"
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

### 🏃‍♂️ Development

To start the Tailwind CSS watcher (which compiles your CSS as you edit):

```bash
npm run tailwind
```

This command runs:
`npx @tailwindcss/cli -i ./src/css/style.css -o ./website/css/style.css --watch --minify`

Open `website/index.html` in your browser to view the site.

## 📂 Project Structure

- `src/`: Contains source files (like the input CSS).
- `website/`: The public-facing distribution folder containing the HTML and compiled CSS.
- `docs/`: Design documents, content drafts, and inspiration resources.
